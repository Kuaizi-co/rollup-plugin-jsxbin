# rollup-plugin-jsxbin

> Plugin for rollup with covert jsx file to jsxbin

## Installation

Install the package:

```
yarn add rollup-plugin-jsxbin -D
// or
npm i rollup-plugin-jsxbin -D
```

## Usage

```js
import jsxbin from 'rollup-plugin-jsxbin'

export default {
  // ...
  plugins: [
    jsxbin()
  ]
}
```
